# Minesweeper
An android implementation of the minesweeper game.

#Features
- [x] Four difficulties.
- [x] Counting Time.
- [x] Set flags.
- [x] Set question marks.
- [x] Enable and disable sounds.
- [x] Smart bot for hints.
- [ ] Record high score.
